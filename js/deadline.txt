PARTE 01 (GITHUB)
 Criar um repo no GH com nome: pokeapi-simple-search
- Você irá criar um pequeno projeto que vai utilizar a mesma API que usamos porém ao invés de pequisar pelo número, vamos pesquisar pelo nome do Pokémon

PARTE 02 (HTML E CSS)
- Sua aplicação terá que ter:
 - Conteúdo todo centralizado na tela
 - Um título: "Procure informações de seu Pokémon favorito!"
 - Um input de texto para que o usuário digite o nome do Pokémon
 - Um botão pesquisar

PARTE 03 (JS)
- Como sua aplicação deve funcionar:
 - O botão de pesquisa deve ficar desabilitado até o usuário digitar pelo menos três caracteres que não seja espaço
 - Ao clicar no botão pesquisar deve enviar um request para a API com o nome digitado no input
 - Se for um pokémon existente vc deve mostrar uma DIV com o número, nome e os tipos do pokémon
 - Se não existir o pokemon procurado vc deve mostrar um alerta para o usuário informando que o pokémon procurado não existe
-  Tecnologias que vc poderá usar:
 - HTML 5, CSS 3, JavaScript (sem usar nenhuma biblioteca ou framework)
- Estrutura de diretório do projeto:
 - Um arquivo index.html
 - Uma pasta css com arquivo(s) css
 - Uma pasta js com arquivo(s) js
 - Uma pasta img com imagem(ns) caso queira
